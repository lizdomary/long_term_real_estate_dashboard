Real Estate Investment Analyzer

website: https://lizortizmunoz.shinyapps.io/realestate/

This is an interactive R Shiny app for analyzing long-term rental property investments. The tool lets you input property details, financing assumptions, and operating expenses to instantly see how metrics like cash flow, Net Operating Income (NOI), Cash-on-Cash Return, Cap Rate, and Debt Service Coverage Ratio (DSCR) change over time.

The app also includes visualizations for yearly cash flow, profit trends, and sensitivity analysis, helping you test how changes in rent, interest rates, property value, or vacancy affect performance. With an integrated advisor feature, you can set profit targets and see what rent levels would be needed to reach them. The goal is to make real estate financial analysis more transparent and accessible for investors.
